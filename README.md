##게임 방식
1. 롤처럼
양 끝에 핵을 두고 핵과 중앙의 가운데 부분에 포탑을 설치
중앙 양 사이드와 포탑 근처에 밖에서 안 보이는 공간을 만듬

2. 눈치게임
중앙에 핵을 둠
핵은 미니언을 때리고 플레이어가 핵을 때리면 내구도가 고정수치로 깍임

3. 파멸의탑
특정 시간마다 점령지가 열리고 일정량의 게이지를 모으면 핵을 공격하는 이벤트를 발생한다.

##이벤트 (다양한 이벤트 중 랜덤으로)
1. 진 팀에게 페널티를 주자.
2. 이긴 팀에게 쿨타임 감소효과를 주자
3. 이긴 팀의 미니언에게 강화 버프를 주자.
